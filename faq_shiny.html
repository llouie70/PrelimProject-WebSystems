Frequently Asked Questions

This page is where your (burning) questions might have an answer here.

Why does this website exist?

This website exists as my Prelim Project for Web Systems and Technologies to Sir Logan Viray. This website aims to help people who cannot decide on a choice or number pick by letting RNG or random number generators decide for them.

What features does this website have?

This website has a dice where you can roll a number between two numbers and a choice picker where you can input your choices and a choice will be displayed on the "Magic 8-Ball."
